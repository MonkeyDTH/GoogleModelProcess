## 项目结构

- Scripts/ - 包含所有Python脚本文件
  - blender_script.py - Blender内部操作脚本
  - merge_mesh.py - 合并场景中所有的Mesh对象
  - mesh_functions.py - 网格处理函数
  - mesh_tools.py - 网格工具
  - open_google_map.py - 打开Google地图并处理
  - osm.py - OSM相关功能
  - render.py - 渲染相关功能
